4S U4 · Plan B para Google Forms

Aquí queda listo el paquete alternativo: imágenes cuadradas publicadas en GitHub Pages y un script de Apps Script que crea un formulario de Google Forms con 12 preguntas adaptadas al contenido exacto de las fichas de estudio.

Script para Apps Script

Copia este código en un proyecto nuevo de Apps Script, guarda y ejecuta la función create4SU4ExamForm. El script crea el formulario como cuestionario e inserta las imágenes desde los links públicos de este repositorio.

Ver archivo fuente
Google Forms no permite algunas mecánicas como arrastrar para ordenar o señalar zonas sobre la imagen. Por eso esas preguntas fueron adaptadas a opción múltiple o casillas, manteniendo el contenido de las fichas.

Qué incluye

  • 12 preguntas alineadas a las cuatro hojas de estudio que compartiste.
  • 4 imágenes cuadradas, listas para insertarse en Google Forms.
  • Preguntas automáticas y autoevaluables en formato quiz.
  • Tipos adaptados: alternativa contextualizada, comparación, secuencia, interpretación de imagen y caso aplicado.

Temas cubiertos: señales y comunicación, neurona, ruta nerviosa, sinapsis, sistema endocrino, ruta hormonal, potencial de membrana y caso de glucosa/insulina.
